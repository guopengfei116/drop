# THREE

## 颜色 Color(color)
- color 16进制颜色

```javascript
let color = new THREE.Color(0x444444);
```

## 场景 Scene()

```javascript
let scene = new THREE.Scene();
```

## 线条 Line(geometry, material, joinType)

#### parameters
- geometry 几何体
- material 线条材质
- joinType 点的连接方式

```javascript
let line = new THREE.Line();
```
