# 概述

- 数据结构：对计算机内存中数据的安排
- 算法：对结构中的数据进行各种处理

应用：

1. 现实世界的数据存储
2. 程序员的工具
3. 现实世界的建模

## 数据结构

| 数据结构 | 优点 | 缺点 |
| -             | -       | -      |
| 数组        | 插入快 | 查找慢，删除慢，大小固定 |
| 有序数组 | 查找快 | 插入慢，删除慢，大小固定 |
| 队列        | 先进先出存取方式 | 存取其他项很慢 |
| 栈           | 先进后出存取方式 | 存取其他项很慢 |
| 链表        | 插入快，删除快 | 查找慢 |
| 二叉树     | 平衡情况下查找，插入，删除都快 | 删除算法复杂 |
| 红黑树     | 查找，插入，删除都快 | 算法复杂 |
| 2-3-4树   | 查找，插入，删除都快 | 算法复杂 |
| 哈希表     | 插入快，有关键字存取快 | 删除慢 |
| 堆            | 插入，删除快，对最大数据项存取很快 | 对其他数据项存取慢 |
| 图            | 对现实世界建模 | 有些算法慢且复杂 |

## Class

Java 是纯面向对象编程的语言，所以有必要重申一下面向对象里的两个基本概念

- 类：许多对象的蓝图，是对象的模型。
- 对象：类的实例，主要包含一些变量或方法
