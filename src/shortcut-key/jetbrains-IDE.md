# JetBrains

JetBrains 是一家捷克的软件开发公司，该公司最为人知的产品是 Java 编程语言开发撰写时所用的集成开发环境：IntelliJ IDEA，还有常见的 PyCharm、PHPStorm、WebStorm 也都出自该公司之手。

## 代码编辑

### 插入

| 功能 | 快捷键 |
|   -   |     -    |
| 插入模板 | Ctrl + J |
| 环绕模板 | Ctrl + Alt + J |

### 补全

| 功能 | 快捷键 |
|   -   |     -    |
| 代码补全 | Ctrl + Space |
| 智能代码补全 | Ctrl + Shift + Space |
