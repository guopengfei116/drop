# Mac

## 终端

#### 光标
- 行首 
  + `Ctrl + a`
- 行尾
  + `Ctrl + e`
- 前进 
  + `Ctrl + f`
- 后退 
  + `Ctrl + b`

#### 编辑
- 粘贴 
  + `Ctrl + y`
- 删除一个字符 
  + `Ctrl + h`
- 删除到行首 
  + `Ctrl + u`
- 清屏 
  + `clear 或 Ctrl + l`

#### 搜索
- 搜索 
  + `Ctrl + r`
- 上个命令 
  + `Ctrl + p`
- 下个命令 
  + `Ctrl + n`

#### 窗口
- 退出 
  + `logout 或 Ctrl + d`
- 新建窗口 
  + `Command + t`
