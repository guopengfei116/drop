# Sublime

## Windows

#### 光标
- 新行行首 
  > Ctrl + 回车
- 上行行首
  > Ctrl + Shift + 回车
- 上移动
  > Ctrl + shift + 上箭头
- 下移动
  > Ctrl + shift + 下箭头
- 行尾
  > Ctrl + e
- 前进 
  > Ctrl + f
- 后退 
  > Ctrl + b

#### 编辑
- 选中多行同时编辑
  > shift + 右键
- 注释
  > Ctrl + /
- 粘贴 
  > Ctrl + v
- 粘贴并保持缩进
  > Ctrl + shift + v
- 复制当前行
  > Ctrl + shift + d
- 删除一个字符 
  > Ctrl + h
- 删除到行首 
  > Ctrl + u
- 清屏 
  > clear 或 Ctrl + l

#### 搜索
- 搜索 
  > Ctrl + r
- 替换
  > Ctrl + h
- 左缩进
  > Ctrl + [
- 右缩进
  > Ctrl + ]
- 跳到指定行
  > Ctrl + g
- 上个命令 
  > Ctrl + p
- 下个命令 
  > Ctrl + n

#### 设置
- 字体大小
  > Ctrl + 滚动

#### 窗口
- 退出 
  > logout 或 Ctrl + d
- 新建窗口 
  > Command + t
